Thesis Project: BleachBoy is a OpenCV based workflow that takes segmented annotated images from CoralVision, an application developed by SJSU professor Philip Heller. 
CoralVision is a Deep Learning Computer Vision model that uses Mask R-CNN with the purpose of being able to identify corals, and coral supported species within images. 
The purpose of the BleachBoy workflow is to identify healthy from bleached coral individuals based on the mean RGB difference between the bleached/healthy coral and the mean RGB of the global image.
